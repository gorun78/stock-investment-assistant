## 🚀 使用方式对比

### 原系统使用方式
```bash
python stock_investment_assistant_main.py
# 交互式输入风险偏好和金额
```

### 专业工具使用方式
```bash
# 1. 基本分析
python stock_pro_tool.py analyze --symbols 300730.SZ,688480.SS --profile 稳健型 --amount 100000

# 2. 配置管理
python stock_pro_tool.py config --set monitoring.interval_minutes 10

# 3. 批量分析
python stock_pro_tool.py analyze --symbols 300730.SZ,688480.SS,300474.SZ --profile 激进型 --amount 50000 --output report.json

# 4. 查看帮助
python stock_pro_tool.py --help
python stock_pro_tool.py analyze --help
```

## ⚠️ 注意事项

### 数据源限制
1. **新浪财经API**
   - 免费但有限制
   - 可能存在延迟
   - 非交易时间不可用

2. **专业数据源**
   - Tushare需要token
   - 东方财富需要配置
   - 建议使用付费数据源

### 网络要求
1. **实时数据**: 需要稳定的网络连接
2. **批量分析**: 建议在低峰时段进行
3. **缓存机制**: 减少重复请求

### 性能优化
1. **并发控制**: 避免API限制
2. **缓存策略**: 合理设置TTL
3. **内存管理**: 及时清理缓存
