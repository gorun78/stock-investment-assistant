## 1. 产品定位

### 1.1 产品愿景

打造一款**专业级股票分析平台**，为个人投资者提供从市场研究、股票分析、投资决策到交易执行的全流程支持，帮助用户做出科学、理性的投资决策。

### 1.2 核心价值主张

| 价值点 | 描述 |
|--------|------|
| 🔍 **深度分析** | 提供基本面、技术面、资金面、消息面四维分析 |
| 📊 **数据驱动** | 基于量化指标和模型进行投资决策 |
| ⚖️ **风险可控** | 内置风险评估和仓位管理机制 |
| 📈 **策略回测** | 支持历史数据验证投资策略 |
| 🎯 **精准选股** | 多策略智能选股，匹配不同投资风格 |

### 1.3 用户画像

| 用户类型 | 投资经验 | 核心需求 | 使用频率 |
|---------|---------|---------|---------|
| **价值投资者** | 3年以上 | 基本面分析、估值模型、长期持有 | 每周1-2次 |
| **趋势交易者** | 2年以上 | 技术分析、趋势跟踪、短线操作 | 每日1-2次 |
| **量化爱好者** | 1年以上 | 策略回测、数据挖掘、模型验证 | 每日多次 |
| **投资新手** | 1年以内 | 学习教程、模拟交易、风险教育 | 每日学习 |

### 2.2 各阶段详细流程

#### 阶段一：市场研究

```
市场研究流程
│
├── 1. 宏观经济分析
│   ├── 查看大盘指数走势
│   ├── 分析市场情绪指标
│   └── 了解政策动态
│
├── 2. 行业板块分析
│   ├── 查看板块涨跌排名
│   ├── 分析板块资金流向
│   └── 识别热点板块
│
├── 3. 自选股票管理
│   ├── 添加关注股票
│   ├── 分组管理自选
│   └── 设置关注提醒
│
└── 4. 智能选股
    ├── 选择选股策略
    ├── 设置筛选条件
    └── 获取推荐股票
```

#### 阶段二：股票分析

```
股票分析流程
│
├── 1. 基本面分析
│   ├── 财务数据查看（营收、利润、ROE、PE等）
│   ├── 估值分析（PE、PB、PEG）
│   └── 成长性分析（营收增长、利润增长）
│
├── 2. 技术面分析
│   ├── K线形态识别
│   ├── 技术指标计算（MACD、RSI、KDJ、布林带）
│   ├── 趋势判断（均线系统）
│   └── 支撑阻力位分析
│
├── 3. 资金面分析
│   ├── 主力资金流向
│   ├── 北向资金动向
│   └── 成交量分析
│
└── 4. 消息面分析
    ├── 公司公告
    ├── 行业新闻
    └── 研报摘要
```

    └── 风险预警设置

## 4. 数据架构

### 4.1 数据实体关系

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   用户配置   │     │   股票信息   │     │   交易记录   │
├─────────────┤     ├─────────────┤     ├─────────────┤
│ user_id     │     │ stock_id    │     │ trade_id    │
│ risk_profile│     │ name        │     │ user_id     │
│ max_position│     │ symbol      │     │ stock_id    │
│ max_drawdown│     │ sector      │     │ type        │
│ theme       │     │ pe/pb/roe   │     │ price       │
└─────────────┘     │ trend       │     │ quantity    │
                    └─────────────┘     │ timestamp   │
                                        └─────────────┘
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   持仓信息   │     │   预警规则   │     │   分析报告   │
├─────────────┤     ├─────────────┤     ├─────────────┤
│ position_id │     │ alert_id    │     │ report_id   │
│ user_id     │     │ user_id     │     │ user_id     │
│ stock_id    │     │ stock_id    │     │ type        │
│ quantity    │     │ alert_type  │     │ content     │
│ avg_cost    │     │ condition   │     │ timestamp   │
│ current_val │     │ is_active   │     └─────────────┘
└─────────────┘     └─────────────┘
```

### 4.2 核心数据字段

#### 股票基础数据

| name | string | 股票名称 | 行情接口 |
| sector | string | 所属板块 | 行情接口 |
| price | float | 当前价格 | 行情接口 |
| change_pct | float | 涨跌幅(%) | 行情接口 |
| volume | float | 成交量 | 行情接口 |
| turnover | float | 成交额 | 行情接口 |
| pe | float | 市盈率 | 财务数据 |
| pb | float | 市净率 | 财务数据 |
| roe | float | 净资产收益率 | 财务数据 |
| market_cap | float | 总市值 | 行情接口 |

#### 技术指标数据

| 字段名 | 类型 | 描述 | 计算方式 |
|--------|------|------|---------|

| kdj_k/d/j | float | KDJ值 | 随机指标 |
| boll_upper/mid/lower | float | 布林带 | 标准差计算 |
