# 界面优化计划

## 问题分析

### 1. 重复展示问题

| 问题 | 位置 | 说明 |
|------|------|------|
| 智能选股功能重复 | 标签页 + 模态框 | 股票维护区域有"智能选股"按钮打开模态框，同时有独立的"智能选股"标签页 |
| 风险偏好设置重复 | 配置表单 + 风险控制 | 投资参数配置和风险控制模块都有风险偏好设置 |

### 2. 展示顺序问题

| 问题 | 说明 |
|------|------|
| 标签页顺序不合理 | 应按用户使用流程排列：自选股票 → 智能选股 → 模拟炒股 → 风险控制 → 投资分析 → 学习中心 |
| 配置表单位置 | 投资参数配置应该在模拟炒股模块内，而不是全局顶部 |

### 3. 界面冗余问题

| 问题 | 说明 |
|------|------|
| 操作按钮位置 | "执行分析"和"生成报告"按钮在配置完成后才能使用，应该移到投资分析模块 |
| 模态框冗余 | 智能选股模态框与标签页功能重复 |

---

## 优化方案

### 方案一：精简整合（推荐）

#### 1. 删除重复功能

- **删除智能选股模态框**：保留独立的"智能选股"标签页，删除股票维护区域的"智能选股"按钮和对应模态框
- **合并风险偏好设置**：将投资参数配置移入模拟炒股模块

#### 2. 优化标签页顺序

```
自选股票 → 智能选股 → 模拟炒股 → 风险控制 → 投资分析 → 学习中心
```

### 方案二：保留但优化

保留现有结构，但做以下调整：
1. 将股票维护区域的"智能选股"按钮改为"快速选股"，简化功能
2. 保留智能选股标签页作为完整功能
3. 添加功能说明区分两个入口

## 推荐方案

**采用方案一：精简整合**

理由：
1. 减少用户困惑，功能入口清晰
2. 界面更简洁，操作更直观
3. 符合用户使用流程：选股 → 模拟 → 分析 → 学习

---

## 实施步骤

1. **删除冗余元素**
   - 删除全局配置表单
   - 删除智能选股模态框
   - 删除股票维护区域的智能选股按钮

2. **调整功能位置**
   - 将投资参数配置移入模拟炒股模块
   - 将分析按钮移入投资分析模块

3. **优化交互逻辑**
   - 更新JavaScript事件绑定
   - 确保功能正常工作

4. **测试验证**
   - 验证所有功能正常
   - 检查界面显示正确
