## 1. 调整概述

### 1.1 调整目标

根据《需求规格说明书 v3.0》，对系统进行以下调整：

1. **新增模块**: 风险控制模块、学习教育模块
2. **功能增强**: 模拟炒股、智能选股、股票自选池
3. **体验优化**: 智能预警、收益曲线、仓位管理

### 1.2 调整原则

- 优先实现高优先级需求
- 保持系统稳定性
- 渐进式开发，逐步迭代
- 每个功能完成后进行测试验证

## 2. 调整计划

### 阶段一：核心功能增强（高优先级）

#### 2.1.1 风险控制模块开发

| 需求编号 | 功能名称 | 实现内容 | 技术方案 |
|---------|---------|---------|---------|
| REQ-031 | 风险测评 | 用户风险承受能力问卷测评 | 前端问卷表单 + 后端评分算法 |
| REQ-032 | 仓位限制提醒 | 监控单一股票仓位占比 | 持仓计算 + 阈值判断 + 提醒 |
| REQ-033 | 止损止盈建议 | 基于技术分析提供价位建议 | ATR/支撑阻力位计算 |
| REQ-034 | 最大回撤监控 | 监控账户最大回撤 | 收益曲线计算 + 回撤检测 |
| REQ-036 | 风险分散建议 | 基于持仓集中度给建议 | 持仓分析 + 分散度计算 |

**后端API设计**:
```
POST /api/risk/profile        - 风险测评
GET  /api/risk/limits         - 获取风险限制配置
PUT  /api/risk/limits         - 更新风险限制配置
GET  /api/risk/alerts         - 获取预警列表
POST /api/risk/alerts         - 设置预警
GET  /api/risk/stop_loss      - 获取止损止盈建议
GET  /api/risk/drawdown       - 获取回撤数据
```

**前端界面**:
- 新增「风险控制」标签页
- 风险测评问卷弹窗
- 仓位分布饼图
- 回撤曲线图表
- 风险预警提示卡片

#### 2.1.3 智能预警提醒功能

| 需求编号 | 功能名称 | 实现内容 | 技术方案 |
|---------|---------|---------|---------|
| REQ-030 | 智能预警提醒 | 价格和指标预警 | 预警规则配置 + 定时检测 |

**预警类型**:
- 价格预警：达到目标价/止损价
- 涨跌幅预警：超过阈值
- 技术指标预警：MACD金叉/死叉、RSI超买超卖
- 仓位预警：单一股票仓位过高

**后端API设计**:
```
GET  /api/alerts              - 获取预警列表
POST /api/alerts              - 创建预警
DELETE /api/alerts/{id}       - 删除预警
GET  /api/alerts/check        - 检查预警触发
```

**前端界面**:
- 预警设置弹窗
- 预警列表展示
- 预警触发通知
