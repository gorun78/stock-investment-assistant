# spec-rfc 技能

最高优先级需求分析技能，从 7 个核心维度挖掘需求，生成结构化规格说明书。

## 触发条件

- 所有新功能开发任务
- 需求变更任务
- 架构调整任务

## 执行流程

```
1. 分析需求描述
2. 识别系统边界
3. 拆解业务能力
4. 记录架构决策
5. 定义技术架构
6. 设计 API 与数据模型
7. 制定执行计划
8. 明确质量标准
```

## 2. Capability Map
